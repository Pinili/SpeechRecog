                     Speech Recognition Engine (SRE)
================================================================================

Installation Instructions
=========================

The following steps are to be performed:

1) Install NetBeans IDE for Java. NetBeans IDE can be downloaded from 
   "http://www.netbeans.org/downloads/". Ensure that the latest version of JDK 
   is installed.
2) Unzip the source environment provided. The source and all relevant files 
   corresponding to the project are extracted to the folder named Speech.
3) Open the project by clicking on File > Open Project and by selecting the 
   Speech folder (source enviroment) obtained in step 2.
4) The project tree displaying the various class files can be viewed in the 
   Project Window (Ctrl + 1) of NetBeans IDE.
5) The entire project is now accessible through NetBeans IDE. The project can 
   be compiled together (F11) and run (F6).

Configuration
=============

The steps for configuration of the system are mentioned in detail in the file
README.
