The following is the list of files included in the package:

Required:

1) The index file          : index.dat
2) The auxiliary files     : *.dat
3) List of training samples: files.dat
4) List of testing samples : test.dat

Files created (Temporary files and Results):

1) cep.dat          : Temporary file. Contains Cepstral coefficients 
2) data.dat         : Temporary file. Contains sample values of speech signals
3) obs.dat          : Temporary file. Contains sample values of test speech signal
4) Codebook.txt     : Result file. Contains the codebook listed for each iteration
5) Distortion.txt   : Contains the overall average distortion computed while LBG algorithm
6) finalCodeBook.txt: The final text codebook
7) final.cbk        : The final binary codebook

Other files:

1) README       : This file
2) INSTRUCTIONS : Steps on how to setup the system

All other files are required by NetBeans IDE for retaining project environment.

Training the SRE:
================

(i) Preparation:
    -----------

The following files are required to start training the speech recognition 
system:

1) "index.dat"
   This file is an index file consisting the names of the auxiliary files and 
   the names of the files to store corresponding HMMs. 
   E.g. the index.dat for recognizing vowels would look as:
        a.dat   a.hmm
        e.dat   e.hmm
        i.dat   i.hmm
        o.dat   o.hmm
        u.dat   u.hmm
   Here, *.dat are the auxiliary files and *.hmm are the names of the 
   corresponding HMMs.
2) The auxiliary files
   These are the files in the first column of "index.dat". Each auxiliary file
   consists of a list of wav files corresponding to the hmm listed in the 
   "index.dat".
   E.g. the auxiliary file a.dat (of the above example) would look as:
        a1.wav
        a2.wav
        a3.wav
        a4.wav
         .
         .
         .
        an.wav
   Here, ai.wav are wav files required to train the SRE to recognize the vowel 
   a.
3) A list of names of the wav files (here, "files.dat") to generate the 
   codebook using vector quantization.

(ii) Training:
     --------

To start training, the method train() in the class Main is to be called with the 
following arguments (in order):
 (a) The name of the codebook file
 (b) The name of the index file
 (c) The name of the file listing all the wav files required for generating 
     the codebook
 (d) Absolute path to the folder containing the files listed in file mentioned
     in section (c) above
 (e) Absolute path to the folder containing the files required to generate
     HMMs

Testing the SRE:
===============

Since speech cannot be fed into the system directly from a speech recorder, the
speech needs to be recorded to a file and then the file is to be fed to the 
system.

(i) Preparation:
    -----------

The following files are necessary to test the SRE:
1) The test file, here, "test.dat"
   Contains the list of wav files to be tested

(ii) Testing:
     -------

To start testing, the method test() in the class Main is called with the 
following arguments (in order):
 (a) The test file, say, "test.dat"
 (b) The directory containing the files mentioned in "test.dat"
 (c) The name of the codebook file
 (d) The name of the index file
 (e) An array of output strings to be displayed on STDOUT when recognized. The
     order of elements in the array should match the order of HMMs listed in 
     the second column on the index file.

** An API for GUI is also available **
